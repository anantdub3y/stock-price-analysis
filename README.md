#  Stock Price Analysis & Forecasting

A data science project analyzing stock market trends and forecasting 
prices using ARIMA modeling.

##  Tools Used
- Python
- yfinance
- Pandas
- Matplotlib & Seaborn
- Statsmodels (ARIMA)

##  Project Highlights
- Stock price visualization with Moving Averages
- Daily Returns & Risk Analysis
- Multi-Stock Comparison & Correlation
- 30-Day ARIMA Price Forecast
